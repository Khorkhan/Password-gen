# Password Generator & Strength Checker

Web app สำหรับสร้างรหัสผ่านและตรวจสอบความแข็งแกร่งของรหัสผ่าน

## ฟีเจอร์

- **สร้างรหัสผ่าน** – เลือกความยาว (8–32 ตัวอักษร), ตัวอักษรใหญ่/เล็ก, ตัวเลข, สัญลักษณ์
- **คัดลอก** – คัดลอกรหัสผ่านที่สร้างไปยังคลิปบอร์ด
- **ตรวจสอบความแข็งแกร่ง** – ใส่รหัสผ่านแล้วดูระดับความแข็งแกร่งและคำแนะนำ

## วิธีรัน

เปิดไฟล์ `index.html` ในเบราว์เซอร์ หรือรันเซิร์ฟเวอร์ท้องถิ่น:

```bash
npx serve .
# หรือ
python -m http.server 8000
```

จากนั้นเปิด http://localhost:3000 (หรือ 8000) ในเบราว์เซอร์
